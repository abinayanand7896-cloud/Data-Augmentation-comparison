# Tabular Data Augmentation Techniques Comparison for Imbalanced classification

**Overview:**

This project is created to compare the data augmentation techniques (SMOTE / ADASYN / CTGAN) for the imbalance classes in Tabular Data. This mainly focus on the binary classification problem with the imbalance ratios 1:10, 1:100, 1:1000

**Datasets used for this project:**

1. Breast cancer:
   https://www.kaggle.com/datasets/reihanenamdari/breast-cancer

2. Credit card fraud:
   https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

**Classifiers Used:**

Logistic Regression, Random Forest, XGBoost

**Instructions for Execution:**
Both the .ipynb files are created through Colab. Please download the datasets from Kaggle, upload the dataset and run the experiment.

If applicable, rename the dataset file name as it is in the program line and execute it.
